# Ement Platform Front-End / Back-End / Database

### Used Technologies:

- [Angular 6](https://angular.io/)
- [ExpressJS Node.js](http://expressjs.com/pt-br/)
- [MySql Database](https://www.mysql.com/)

### Project Specifications: 

- Project should have Front End done by Angular 6 and Backend done by some recommended technology (it was not in specification, but ExpressJS was adopted for Backend), and database (MySql).
- Perform the registration of a user using the data people passed through the view.
- Recever the personal data and save in the database through the controller and model.
- Allow user login and display data on another screen.

### How to Setup the Project Environment: </h3>

1. Clone or download the repository. Then go and install node modules to the front-end and back-end folders by using below command. 
    ```
    npm install
    ``` 
2. Run the xampp and build up the **ement_db** database using sql file provided.
3. Go to the **back-end** folder by a terminal and execute the below command to run the server. Keep the terminal. 
    ```
    node server.js
    ``` 
4. Go to the **front-end** folder by another terminal and execute the below command to run the GUI. Keep the terminal. 
    ```
    ng serve --open
    ```
